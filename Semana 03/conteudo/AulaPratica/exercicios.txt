1. Crie uma função que recebe uma string 
(com quatro possibilidades: “soma”, “subtrai”, “divide”, “multiplica”) e
 dois números. 
 A função deve retornar a operação realizada 
 informada pela string nos dois números.

Exemplo Entrada:
myFunction(‘soma’)(3)(5)

Exemplo Saída:
8

2. Crie uma função que recebe um objeto usuário e 
imprime se ele é maior de idade, 
utilize os conhecimentos da aula para melhorar o código.

Exemplo Entrada:
{ name: ‘Juca’, idade: 28 }

Exemplo Saída:
Maior de idade
